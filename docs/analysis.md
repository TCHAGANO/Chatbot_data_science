# PHASE 2 — Analyse des données

Lors de cette étape, j’ai réalisé une première exploration des données (EDA — Exploratory Data Analysis) afin de comprendre la structure de mon dataset avant de construire le chatbot. J’ai utilisé Pandas pour charger le fichier CSV et vérifier que les données étaient correctement importées. J’ai affiché un aperçu des premières lignes (`head()`), identifié les noms des colonnes (`columns`), vérifié les types de données et la présence éventuelle de valeurs manquantes (`info()`), puis observé des statistiques globales (`describe()`). Cette étape est importante car elle permet de détecter des erreurs, comprendre les relations entre les données et éviter des problèmes plus tard dans la création de la base PostgreSQL, des requêtes SQL et du chatbot.
